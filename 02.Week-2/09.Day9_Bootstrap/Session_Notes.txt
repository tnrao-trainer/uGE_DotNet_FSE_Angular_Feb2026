What is debugging?

--> Debugging in JavaScript is the process of finding and fixing mistakes (bugs) or unexpected behavior in code. Since errors are common in software development, debugging is an essential skill for any JavaScript developer.





Javascript frameworks/libraries
~~~~~~~~~~~~~~~~~~~~~~~~~~~~


--> JavaScript frameworks and libraries are collections of pre-written, reusable code that help developers build dynamic and complex web applications more efficiently by providing structure and pre-built functionalities. 

--> The key technical difference is inversion of control: you call a library's code, while a framework calls your code within its predefined structure. 


eg:   jQuery, React ,  Angular, Vue etc...
