What is Assembly in .NET?

In .NET, an assembly is a fundamental building block, serving as a compiled unit of code that the Common Language Runtime (CLR) executes. It is the smallest unit of deployment, versioning, reuse, and security in a .NET application. Assemblies are implemented as either executable files (.exe) or dynamic link libraries (.dll).


What is GAC?

The Global Assembly Cache (GAC) in .NET is a machine-wide, centralized repository (located in %windir%\Microsoft.NET\assembly) that stores shared, strongly-named .NET assemblies, enabling code reuse across multiple applications and preventing "DLL hell". It facilitates side-by-side versioning, allowing different versions of the same DLL to coexist.

int.TryParse()
~~~~~~~~~~~
The TryParse method in C# is a static method used to safely convert a string representation of a value into a specific data type without throwing an exception if the conversion fails. Instead of an exception, it returns a boolean value indicating whether the conversion was successful, making it ideal for handling user input or external data where the format is not guaranteed.

What is boxing and unboxing?


Boxing in C# converts a value type (e.g., int, struct) to type object (reference type) on the heap, while unboxing converts it back to a value type.

Boxing is implicit, unboxing is explicit and requires a cast. 

These processes allow value types to be treated as objects but introduce performance overhead due to memory allocation.
